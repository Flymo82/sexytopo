SexyTopo Guide

Overview

SexyTopo is an app for surveying caves and underground spaces. It is designed to work with a DistoX or DistoX2 measuring tool which the surveyor uses to measure out survey legs. The app communicates with the DistoX wirelessly over Bluetooth and automatically creates a model of the cave from measurements made with the device.

Setup

SexyTopo runs on Android devices (the newer/more powerful the better). For obvious reasons a water- and dust-proof device is recommended. It is possible to control SexyTopo with finger movements but it is designed for use with a stylus.

Sketching

The sketch views (plan and extended elevation) can be accessed through buttons on the Action Bar at the top of the screen. These show a projection of the calculated survey on a canvas so that a sketch of the cave can be drawn on top.

Several sketching tools are provided via a toolbar at the bottom of the screen:

  1. Colours set the current sketching colour
  2. Zoom Out
  3. Zoom In
  4. Move
  5. Draw
  6. Erase deletes lines sketched in Draw mode
  7. Settings opens an extended commands menu
  8. Undo
  9. Redo
